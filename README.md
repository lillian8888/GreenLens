## Installation

### Step 1: Get Your Claude API Key

1. Go to https://console.anthropic.com
2. Sign up or log in
3. Navigate to API Keys section
4. Create a new API key
5. Copy the key (starts with `sk-ant-...`)

### Step 2: Install the Extension

1. Download all the extension files to a folder
2. Open Chrome and go to `chrome://extensions`
3. Turn on "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the folder containing the extension files
6. The extension icon should appear in your toolbar!

### Step 3: Configure API Key

1. Click the extension icon (green leaf)
2. Paste your Claude API key in the input field
3. Click "Save API Key"

## How to Use

1. Visit any product page (Amazon, eBay, etc.)
2. Click the extension icon
3. Click "Analyze Current Page"
4. Wait a few seconds for Claude to analyze
5. View your sustainability score and detailed analysis!

## Troubleshooting

**"Could not detect product information"**
- Make sure you're on an actual product page (not a category or search page)
- Try refreshing the page and clicking analyze again

**"API request failed"**
- Check that your API key is correct
- Verify you have credits in your Anthropic account
- Check your internet connection

**Score seems inaccurate?**
- Claude analyzes based on available information on the page
- Limited product details = less accurate analysis
- Consider it as a helpful guide, not absolute truth
